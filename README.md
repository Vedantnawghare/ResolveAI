AgriChain – Farm-to-Market Intelligence Platform

AgriChain is an AI-driven advisory system that helps farmers:

• Predict crop prices
• Choose the best mandi
• Minimize spoilage risk
• Understand why recommendations are made

Tech Stack
• Next.js
• TypeScript
• Tailwind CSS

=> Run Locally
npm install
npm run dev


Data Source

Agmarknet (Government of India) – 3 Month Aggregated Modal Prices
Integrated with live OpenWeather API

That boosts credibility instantly.
